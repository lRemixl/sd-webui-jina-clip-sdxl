# Jina CLIP v2 SDXL Adapter for Forge Neo

## Fully vibe-coded

This extension adds support for jina-clip-v2 adapter as a text encoder for Mugen.
- [jina-clip-v2](https://huggingface.co/jinaai/jina-clip-v2)
- [Adapter+UNet](https://huggingface.co/TheRemixer/Mugen-Jina-V1.5)

Jina-clip-v2 LLM model should be downloaded to `models/llm/` or `models/text_encoder/`

The jina-clip-v2 adapter should be downloaded to `models/llm_adapter/`
